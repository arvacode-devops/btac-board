Step 1 (Verify Card)
vdsl profile show
  OR
adsl profile show

Step 2 (Verify Model)
sys info show

Step 3 (Set DSL Port)
vdsl enable $1ST_PORT#
vdsl name $1ST_PORT# $Customer_Name
vdsl tel $1ST_PORT# $CUSTOMER_TEL_NUMBER
vdsl rate mode $1ST_PORT# profile
vdsl profile map $1ST_PORT# $BW_PROFILE $service_type 
vdsl transfer-mode set $1ST_PORT# atm

Step 4 (CrossConectt)
vdsl dt set $1ST_PORT# $C-TAG 0 $V-TAG 0 
vdsl dt enable $1ST_PORT#

vdsl dtpvc set $1ST_PORT# 0 35 DEFVAL $C-TAG 0 $V-TAG 0 auto llc  
0 35 are VPI, VCI values coming from DSL modems

Step 5 (Save)
 config save
 
Step 6 (Verify)
vdsl show $1ST_PORT#
 OR
adsl show $1ST_PORT#

Verify Cross Connect Provisioning
Show DT connection
vdsl dt show $1ST_PORT#
Show DTPVC connection
vdsl dtpvc show $1ST_PORT#
