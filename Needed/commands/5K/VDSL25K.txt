Step 1 (Setting QoS)

$DOWNRATEkbps-down or UpToMaxSpeedFortheRangeProfileMbps 

Example: 14976kbps-down for “Fixed” (non-ranged) QoS Format and  Ranged QoS Format:  UpTo36Mbps

Step 2 (Verifications)

Check VLAN: show table evc
Check neighbors: sh tab evc-map $shelf/slot sub
Check for a group: sh efm-group $shelf/slot

Step 3 (Provisioning)

Config  t
interface vdsl $SHELF/SLOT/PORT
 description $CUSTOMER_TEL_NUMBER
 deployment region 1 scenario 2 other-service 1 application-standard 3 band-plan 1 band-profile $BAND_PROFILE_7/2 psd-u0 1 psd-mask 1
 dmt-options snr-margin downstream margins minimum 2.0 target 6.0 maximum 16.0
 dmt-options snr-margin upstream margins minimum 2.0 target 6.0 maximum 16.0
 latency-path delay downstream 0 5
 latency-path delay upstream 0 5
 latency-path inp downstream 0 0.5
 latency-path inp upstream 0 0.5
 latency-path payload-rate downstream 0 minimum 128 maximum $DOWNSTREAM_RATE
 latency-path payload-rate upstream 0 minimum 32 maximum $UPSTREAM_RATE   
 latency-path type downstream 0 interleave
 latency-path type upstream 0 interleave
 latency-path retransmission downstream 0 mode custom
 latency-path retransmission downstream 0 custom-config delay 0-20
 latency-path retransmission downstream 0 custom-config max-ndr $DOWNSTREAM_RATE
 latency-path retransmission upstream 0 mode custom
 latency-path retransmission upstream 0 custom-config delay 0-20
 latency-path retransmission upstream 0 custom-config max-ndr $UPSTREAM_RATE
 power-back-off upstream mode disable
 rate-adaption down-shift downstream margin 3.0
 rate-adaption down-shift downstream time 60
 rate-adaption down-shift upstream margin 3.0
 rate-adaption down-shift upstream time 60
 rate-adaption up-shift downstream margin 9.0
 rate-adaption up-shift downstream time 60

 no shutdown
exit

interface atm-port $SHELF/SLOT/PORT
shutdown
